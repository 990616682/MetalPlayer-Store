Before Contacting Support

To help us understand the issue more quickly, please include the following information when possible:

  • App name and version
  • Device model
  • Operating system version
  • A short description of the issue
  • Steps to reproduce the issue
  • Screenshots or screen recordings, if helpful

Common Issues

Video processing is slow

Video frame generation and enhancement can be computationally intensive. Processing speed depends on your device model, video resolution, selected quality mode, and output settings.

The exported video is larger than expected

Output file size depends on the selected codec, bitrate, resolution, frame rate, and source video content. Higher quality settings may produce larger files.

The app cannot open a specific video

Some video containers, codecs, subtitles, or audio formats may not be fully supported. If a file cannot be opened correctly, please contact us with the file format and codec information if available.

常见问题

视频处理速度较慢

视频帧生成和画质增强通常需要较高的计算性能。处理速度会受到设备型号、视频分辨率、质量模式和导出设置的影响。

导出后的视频文件较大

输出文件大小取决于编码格式、码率、分辨率、帧率以及源视频内容。更高质量的设置可能会生成更大的文件。

App 无法打开某些视频

部分视频封装格式、编码格式、字幕格式或音频格式可能无法完全支持。如果某个文件无法正常打开，请尽量提供文件格式和编码信息。

欢迎反馈功能建议。请尽量说明你希望增加的功能，以及它对你的使用流程有什么帮助。

隐私
